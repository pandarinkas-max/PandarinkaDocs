**Drop to Surface** drops selected Studio objects onto the nearest valid surface below them.

## Quick Start

1. Select one or more Studio objects.
2. Open **Pandarinka Toolkit**.
3. Go to **Transform**.
4. Click **Drop Selected to Surface**.

Default shortcut:

```text
U
```
