# Copy Pose From

**Copy Pose From** copies the current pose from one character to other selected characters.

Use it when you have one character posed correctly and want another character to take the same pose quickly.

## Quick Start

1. Open **Pandarinka Toolkit**.
2. Go to **Characters**.
3. Select the character with the pose you want to copy.
4. Click **Set Selected As Source**.
5. Select one or more target characters.
6. Click **Copy Pose From Source**.

The target character receives the source pose.

If the source character is in IK mode, the plugin copies the IK pose. Otherwise, it copies the FK pose.

## Notes

- The source character is not changed.
- You can copy the pose to more than one target character at once.
- The copy action can be undone with **Ctrl + Z**.
