# Pandarinka Docs

Short guides for Pandarinka Studio tools.

## Current Guides

- [Pandarinka Toolkit](pandarinka-toolkit.md) - Studio utility plugin with multiple tools in one window.
  - <a class="program-link" href="#/pose-match">PoseMatch Program</a> - desktop program for creating pose drafts from reference images.
  - [Pose Match](pose-match.md) - apply pose drafts to selected Studio characters.
  - [Scatter Objects](scatter-objects.md) - duplicate objects across an area or surface.
  - [Bake to Mesh](bake-to-mesh.md) - combine a generated folder into a baked mesh object.
  - [Drop to Surface](drop-to-surface.md) - drop selected objects onto the nearest surface below.
  - [Keep Texture Scale](keep-texture-scale.md) - keep texture tiling consistent while objects are scaled.
  - [Joint Follow](joint-follow.md) - keep IK elbows and knees aligned while moving hands or feet.
  - [Options](toolkit-options.md) - hotkeys, window size, position, and opacity.
- [Quick Pick](quick-pick.md) - select Studio objects from the viewport, extract map objects, cache assets, and cache full maps.
- [Studio FBX Exporter](fbx-exporter.md) - export selected Studio objects or characters to FBX with textures.
- [Scene Browser Pro](scene-browser-pro.md) - browse scenes faster and import only selected scene parts.
- [Recycle Bin](recycle-bin.md) - restore deleted objects or stash scene parts for later.
